# Installation Guide for Dawn Sections Pack – Beauty

## Overview
This package contains everything a customer needs to install the **Dawn Sections Pack – Beauty** theme on a Shopify store.

## Contents
- `products/dawn-sections-pack-beauty/` – Full theme source (sections, assets, templates, docs, marketing assets, previews).
- `shared/` – Reusable snippets, CSS, JS, fonts, and other assets.
- `docs/` – Documentation and customization guide.
- `README.md` – High‑level overview of the product.
- `INSTALL.md` – This file.

## Installation Steps
1. **Download the zip** (the latest release) from the GitHub releases page, e.g.:
   ```
   https://github.com/Devhoss/shopify-gumroad-products/releases/download/v1.0.0/dawn-sections-pack-beauty-v1.0.0.zip
   ```
2. **Extract** the zip to a local folder.
3. **Upload to Shopify**:
   - In your Shopify admin, go to **Online Store → Themes → Upload theme**.
   - Choose the extracted folder (the zip contains the theme root).
   - Click **Upload**.
4. **Preview** the theme:
   - From the Themes page, click **Actions → Preview**.
   - Verify layout, sections, and assets render correctly.
5. **Publish** (optional):
   - When satisfied, click **Actions → Publish** to make it live.
6. **Customization**:
   - Edit any section file in `sections/` to match your branding.
   - Use the `docs/` folder for detailed customization instructions.
