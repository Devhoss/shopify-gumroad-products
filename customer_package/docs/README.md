# Shopify Dawn‑Compatible Premium Sections Pack

**Audience:** Perfume & beauty stores looking for a high‑end, performance‑first theme extension.

## Overview
- 25 fully‑featured Liquid sections (perfume, skincare, makeup, accessories, gifts, education, events, community, support).
- Designed for the **Shopify Dawn** theme – one‑click install (copy the `sections/` folder into your Dawn theme).
- Luxury aesthetic: navy #0A0A2A, gold #D4AF37, cream #F9F9F9.
- Minimal CSS/JS for fast load times.

## Installation
1. Download the release zip from the **Releases** page (or run `tools/build-zip.sh`).
2. Unzip into your theme folder:
   ```
   cp -R sections/ <your‑theme>/sections/
   cp -R snippets/ <your‑theme>/snippets/
   cp -R assets/ <your‑theme>/assets/
   ```
3. In the Shopify admin, go to **Online Store → Themes → Customize** and add the sections where desired.

## Features
- **Responsive design** – mobile, tablet, desktop.
- **Schema‑driven settings** – colour pickers, image pickers, text fields, CTA links.
- **Reusable components** – button, icons, colour palette.
- **Performance‑first** – only essential CSS, optional JS for animations.
